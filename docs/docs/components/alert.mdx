# Alert

Alerts are used to display a message to the user. They are modal, meaning that they block interaction with the rest of the page until the user closes the alert.

## Basic usage

<button onClick={(e) => document.querySelector('#myalert').open(e.target)} className="button button--primary">Open alert dialog</button>

<bl-alert-dialog title="Are you sure?" id="myalert" cancel=".cancel" class="blur">
  <div>Are you sure you want to do this? This is permanent!</div>
  <div style={{display: 'flex', justifyContent: 'flex-end', marginTop: '1rem', gap: '0.5rem'}}>
    <button className="button button--secondary cancel">No</button>
    <button className="button button--primary">Yes</button>
  </div>
</bl-alert-dialog>

<br />
<br />

```html
<button data-alert="#myalert">Open alert dialog</button>

<bl-alert-dialog title="Are you sure?" id="myalert" cancel=".cancel" class="blur">
  <div>Are you sure you want to do this? This is permanent!</div>
  <div>
    <button>No</button>
    <button>Yes</button>
  </div>
</bl-alert-dialog>
```
