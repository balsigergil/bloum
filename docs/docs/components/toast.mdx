# Toast

## Basic usage

<div>
  <button className="button button--primary" onClick={() => {Toast.info('This is an info toast')}}>Show info toast</button>
</div>

<br />

First, you need to add the `bl-toast-container` component to your page. This component will be used to display the toasts.

```html
<bl-toast-container></bl-toast-container>
```

Then, you can use the `Toast` class to display toasts.

```javascript
import { Toast } from 'bloum';
Toast.info('This is an info toast');
```

You can also use the `Toast` class directly in your HTML because it is exposed in the global scope.

```html
<button onclick="Toast.info('This is an info toast')">
  Show info toast
</button>
<bl-toast-container></bl-toast-container>
```




## API

### `ToastOptions`

Use the `ToastOptions` interface to customize the toasts:

| Name          | Type       | Default     | Description                                                              |
|---------------|------------|-------------|--------------------------------------------------------------------------|
| `title`       | `string`   | `undefined` | The title of the toast.                                                  |
| `description` | `string`   | `undefined` | An optional description of the toast.                                    |
| `isClosable`  | `boolean`  | `false`     | Whether the toast can be dismissed by the user or not.                   |
| `duration`    | `number`   | `8000`      | The duration of the toast in milliseconds.                               |
| `status`      | `string`   | `info`      | The status of the toast. Can be `info`, `success`, `warning` or `error`. |
| `onClose`     | `function` | `undefined` | A callback function that will be called when the toast is dismissed.     |

### `Toast.info(title, options)`

Display an info toast.

| Name      | Type           | Default     | Description               |
|-----------|----------------|-------------|---------------------------|
| `title`   | `string`       | `undefined` | The message of the toast. |
| `options` | `ToastOptions` | `undefined` | The options of the toast. |

### `Toast.success(title, options)`

Display a success toast.

| Name      | Type           | Default     | Description               |
|-----------|----------------|-------------|---------------------------|
| `title`   | `string`       | `undefined` | The message of the toast. |
| `options` | `ToastOptions` | `undefined` | The options of the toast. |

### `Toast.warning(title, options)`

Display a warning toast.

| Name      | Type           | Default     | Description               |
|-----------|----------------|-------------|---------------------------|
| `title`   | `string`       | `undefined` | The message of the toast. |

### `Toast.error(title, options)`

Display an error toast.

| Name      | Type           | Default     | Description               |
|-----------|----------------|-------------|---------------------------|
| `title`   | `string`       | `undefined` | The message of the toast. |

### `Toast.show(options)`

Display a toast.

| Name      | Type           | Default     | Description               |
|-----------|----------------|-------------|---------------------------|
| `options` | `ToastOptions` | `undefined` | The options of the toast. |
