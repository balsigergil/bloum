# Tokens

Tokens are the visual design atoms of Bloum's design system. They're named entities that store visual design attributes, which help maintain consistency across your application.

## What are Tokens?

Design tokens are CSS custom properties (variables) that define the core visual properties of your UI. They include colors, typography, spacing, shadows, and more.

## Available tokens

All Bloum tokens are prefixed with `--bl-` to avoid conflicts with other CSS variables in your project.

### Core tokens

#### Text and background colors

```css
/* Light mode (default) */
--bl-clr-text: var(--bl-clr-gray-950);
--bl-clr-text-inverted: var(--bl-clr-white);
--bl-clr-background: var(--bl-clr-gray-50);
--bl-clr-surface: var(--bl-clr-white);
--bl-border-color: var(--bl-clr-gray-200);

/* Dark mode (automatic) */
.dark {
  --bl-clr-text: var(--bl-clr-white);
  --bl-clr-text-inverted: var(--bl-clr-gray-950);
  --bl-clr-background: var(--bl-clr-gray-950);
  --bl-clr-surface: var(--bl-clr-gray-900);
  --bl-border-color: var(--bl-clr-gray-800);
}
```

### Layout Tokens

#### Border Radius

```css
--bl-border-radius: 0.25rem;      /* 4px - Default */
--bl-border-radius-md: 0.375rem;  /* 6px - Medium */
--bl-border-radius-lg: 0.5rem;    /* 8px - Large */
--bl-border-radius-xl: 0.75rem;   /* 12px - Extra large */
--bl-border-radius-2xl: 1rem;     /* 16px - 2x large */
--bl-border-radius-full: 9999px;  /* Full circle */
```

#### Shadows

```css
--bl-shadow-sm: 0 1px 2px 0 rgb(0 0 0 / 0.05);
--bl-shadow: 0 4px 6px -1px rgb(0 0 0 / 0.1), 
             0 2px 4px -2px rgb(0 0 0 / 0.1);
--bl-shadow-lg: 0 10px 15px -3px rgb(0 0 0 / 0.1), 
                0 4px 6px -4px rgb(0 0 0 / 0.1);
--bl-shadow-xl: 0 20px 25px -5px rgb(0 0 0 / 0.1), 
                0 8px 10px -6px rgb(0 0 0 / 0.1);
```

### Animation Tokens

#### Transition Durations

```css
--bl-transition-xs: 50ms;   /* Extra small */
--bl-transition-sm: 100ms;  /* Small */
--bl-transition-md: 150ms;  /* Medium */
--bl-transition-lg: 200ms;  /* Large */
--bl-transition-xl: 300ms;  /* Extra large */
--bl-spinner-speed: 0.5s;   /* Spinner animation */
```

### Typography Tokens

#### Font Families

```css
--bl-font-sans: ui-sans-serif, system-ui, sans-serif, 
                "Apple Color Emoji", "Segoe UI Emoji", 
                "Segoe UI Symbol", "Noto Color Emoji";
--bl-font-serif: ui-serif, Georgia, Cambria, 
                 "Times New Roman", Times, serif;
--bl-font-mono: ui-monospace, SFMono-Regular, Menlo, 
                Monaco, Consolas, "Liberation Mono", 
                "Courier New", monospace;
--bl-font-body: var(--bl-font-sans); /* Default body font */
```

### Responsive Breakpoints

```css
--bl-breakpoint-sm: 640px;   /* Small devices */
--bl-breakpoint-md: 768px;   /* Medium devices */
--bl-breakpoint-lg: 1024px;  /* Large devices */
--bl-breakpoint-xl: 1280px;  /* Extra large devices */
--bl-breakpoint-2xl: 1536px; /* 2X large devices */
```

### Form Tokens

```css
--bl-form-control-height: 2.25rem;                      /* 36px */
--bl-form-control-padding: 0.625rem;                    /* 10px */
--bl-form-control-border-color: var(--bl-clr-gray-300); /* Light mode */
--bl-form-control-border-radius: var(--bl-border-radius-md);
--bl-form-control-bg: var(--bl-clr-surface);

/* Dark mode */
.dark {
  --bl-form-control-border-color: var(--bl-clr-gray-700);
}
```

### Interactive States

#### Focus

```css
--bl-focus-color: var(--bl-clr-primary-600); /* Light mode */
--bl-focus-width: 0.2rem;                     /* 3.2px */

.dark {
  --bl-focus-color: var(--bl-clr-primary-500); /* Dark mode */
}
```

#### Disabled State

```css
--bl-disabled-opacity: 0.7;
```

## Customizing Design Tokens

You can customize Bloum by overriding tokens at the root level.

```css
/* Global customization */
:root {
  /* Change primary color to purple */
  --bl-clr-primary-500: #7c3aed;
  --bl-clr-primary-600: #6d28d9;
  
  /* Increase default border radius */
  --bl-border-radius: 0.5rem;
  
  /* Use different font */
  --bl-font-body: 'Inter', var(--bl-font-sans);
}

/* Component-specific customization */
.my-section {
  --bl-clr-surface: #f8f9fa;
  --bl-border-color: #dee2e6;
}
```
