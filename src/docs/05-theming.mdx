# Theming

Bloum is built with customization in mind. Using CSS custom properties (variables), you can easily adapt the design system to match your brand and design requirements.

## How Theming Works

Bloum uses CSS custom properties for all design decisions. This approach provides:

- **Runtime theming** - Change themes without rebuilding
- **Scoped customization** - Override styles for specific sections
- **Browser DevTools support** - Inspect and modify variables live
- **Inheritance** - Child elements automatically inherit theme changes

## Quick Start

The simplest way to customize Bloum is to override CSS variables in your stylesheet:

```css
/* Override primary color globally */
:root {
  --bl-clr-primary-500: #7c3aed;  /* Purple instead of blue */
  --bl-clr-primary-600: #6d28d9;
}

/* Increase default border radius */
:root {
  --bl-border-radius: 0.5rem;     /* 8px instead of 4px */
}

/* Use a custom font */
:root {
  --bl-font-body: 'Inter', var(--bl-font-sans);
}
```

## Creating a Custom Theme

### Step 1: Define Your Theme

Create a CSS file for your theme:

```css
/* my-theme.css */
:root {
  /* Brand Colors */
  --bl-clr-primary-50: #f3e8ff;
  --bl-clr-primary-100: #e9d5ff;
  --bl-clr-primary-200: #d8b4fe;
  --bl-clr-primary-300: #c084fc;
  --bl-clr-primary-400: #a855f7;
  --bl-clr-primary-500: #9333ea;  /* Main brand color */
  --bl-clr-primary-600: #7e22ce;
  --bl-clr-primary-700: #6b21a8;
  --bl-clr-primary-800: #581c87;
  --bl-clr-primary-900: #3b0764;
  --bl-clr-primary-950: #2a0845;
  
  /* Typography */
  --bl-font-body: 'Poppins', var(--bl-font-sans);
  
  /* Spacing & Layout */
  --bl-border-radius: 0.75rem;
  --bl-border-radius-md: 1rem;
  --bl-border-radius-lg: 1.25rem;
  
  /* Shadows - Softer for modern look */
  --bl-shadow: 0 4px 6px -1px rgb(0 0 0 / 0.05), 
               0 2px 4px -2px rgb(0 0 0 / 0.05);
}
```

### Step 2: Load Your Theme

Include your theme after Bloum's CSS:

```html
<!-- Bloum base styles -->
<link href="https://unpkg.com/bloum/dist/bloum.min.css" rel="stylesheet">

<!-- Your custom theme -->
<link href="/css/my-theme.css" rel="stylesheet">
```


## Scoped Themes

Apply different themes to different sections:

```css
/* Main theme */
:root {
  --bl-clr-primary-500: #3b82f6;  /* Blue */
}

/* Premium section with purple theme */
.premium-section {
  --bl-clr-primary-500: #9333ea;  /* Purple */
  --bl-clr-surface: #faf5ff;      /* Light purple surface */
}

/* Dark section */
.dark-section {
  --bl-clr-text: var(--bl-clr-white);
  --bl-clr-background: var(--bl-clr-gray-900);
  --bl-clr-surface: var(--bl-clr-gray-800);
}
```
