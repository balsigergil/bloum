# Installation

Bloum can be installed in multiple ways depending on your project setup and preferences.

## Quick Start with CDN

The fastest way to get started with Bloum is using a CDN. Add these two lines to your HTML `<head>`:

```html
<!-- Bloum CSS -->
<link href="https://unpkg.com/bloum/dist/bloum.min.css" rel="stylesheet">

<!-- Bloum JavaScript -->
<script src="https://unpkg.com/bloum" defer></script>
```

That's it! You can now start using Bloum components in your HTML.

> The CDN version of Bloum is not recommended for production use.

### Specific Version

To use a specific version of Bloum (recommended for production):

```html
<!-- Replace X.Y.Z with the desired version -->
<link href="https://unpkg.com/bloum@X.Y.Z/dist/bloum.min.css" rel="stylesheet">
<script src="https://unpkg.com/bloum@X.Y.Z" defer></script>
```

## Package Manager Installation

For modern JavaScript projects using build tools, install Bloum via your preferred package manager:

```bash
# npm
npm install bloum

# yarn
yarn add bloum

# pnpm
pnpm add bloum

# bun
bun add bloum
```

## Importing in Your Project

After installation, import Bloum in your JavaScript entry point:

### With a Module Bundler

If you're using Vite, Webpack, Rollup, or another bundler:

```javascript
// main.js or app.js
import "bloum"

// Import Bloum styles
import "bloum/dist/bloum.min.css"
```

## Verify Installation

To verify Bloum is properly installed, add this button to your HTML:

```html
<button class="btn btn-primary">
  Hello Bloum!
</button>
```

If you see a styled button, congratulations! Bloum is ready to use.
