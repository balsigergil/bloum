# Color System

Bloum provides a comprehensive color palette inspired by Tailwind CSS, with semantic color mappings for consistent theming across your application.

## Semantic Color Palette

Bloum includes a full spectrum of colors, each with 11 shades ranging from 50 (lightest) to 950 (darkest). All colors are available as CSS custom properties.

You can customize the colors by overriding the CSS variables.

### Primary

The primary color is used for main actions, links, and focus states.

<Palette colorName="primary" />

```css
:root {
  --bl-clr-primary-50: var(--bl-clr-blue-50);
  --bl-clr-primary-100: var(--bl-clr-blue-100);
  --bl-clr-primary-200: var(--bl-clr-blue-200);
  --bl-clr-primary-300: var(--bl-clr-blue-300);
  --bl-clr-primary-400: var(--bl-clr-blue-400);
  --bl-clr-primary-500: var(--bl-clr-blue-500);

  --bl-clr-primary-900: var(--bl-clr-blue-900);
  --bl-clr-primary-950: var(--bl-clr-blue-950);
}
```
